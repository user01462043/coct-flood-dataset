Cape Town Flood WebGIS v5 - app-style section views

This version no longer behaves like a long report page.
Use the top menu or the homepage cards to open one section at a time:
- Home / website contents
- About
- Framework
- Methodology
- Limitations
- Engage with data

Run:
1. Open the extracted folder in VS Code.
2. Right-click index.html.
3. Open with Live Server.

Edit:
- index.html = text and section layout
- assets/style.css = appearance
- assets/app.js = interactions, map, charts and downloads
- data/events.json = flood records used by the site
